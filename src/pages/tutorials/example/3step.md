3. MySQL and MySQL Workbench
    - **MySQL**
        - In the terminal:
        ```js
        '$ brew install mysql'
        ```
        - More info: *[SQL](https://gist.github.com/nrollr/3f57fc15ded7dddddcc4e82fe137b58e)*
        
    - **MySQL Workbench**
        - In the terminal:

        ```js
        'brew tap caskroom/cask'
        'brew install brew-cask'
        'brew cask install mysqlworkbench'       
         ```
